MAKING THIS WEBSITE

I made this website with the goal of gaining a better understanding of front end development. I used HTML, CSS, and some basic JavaScript to create this website. I really enjoyed the process of starting from scratch and having to teach something completely new to myself. I now have a basic understanding of how websites are created and coding concepts. This project is only the beginning of my curiousity into the tech space!

INSIGHTS

  • This project helped me understand what resources to use - such as Github Pages for hosting the website, Stack Overflow for when I had questions, and Youtube for a many tutorials!

  • Learning HTML and CSS helped me understand design, layout, coding concepts, and file structure.

  • I had to understand how the layout would be impacted on different device screen size, such as desktop vs mobile. I used VS Code extensions to test my design on Iphones while I worked primarily on my computer.
